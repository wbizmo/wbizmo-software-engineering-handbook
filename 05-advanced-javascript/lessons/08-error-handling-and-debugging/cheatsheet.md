# Lesson 08 Cheatsheet

Syntax error:
    Code is written incorrectly.

Reference error:
    Variable does not exist.

Type error:
    Wrong value usage.

try catch:

    try {
      // risky code
    } catch (error) {
      console.log(error.message);
    }

throw:

    throw new Error("Message");

Debug:

    console.log(value);

Error message clues:

    Type
    Message
    File
    Line
    Stack trace
