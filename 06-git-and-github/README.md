# Git and GitHub

## Overview

Git is the most widely used version control system in software development.

GitHub is the most popular platform for hosting Git repositories, collaborating with teams, reviewing code, managing projects, and showcasing engineering work.

This module teaches version control from beginner to professional workflows.

## Why This Module Matters

Professional software development depends on version control.

Without Git:

- Collaboration becomes difficult
- History is lost
- Rollbacks become painful
- Team workflows break down

Every developer is expected to know Git.

## Learning Outcomes

By the end of this module you will be able to:

- Initialize repositories
- Create commits
- Use branches
- Merge code
- Resolve conflicts
- Push and pull from GitHub
- Use pull requests
- Manage releases and tags
- Follow professional Git workflows
