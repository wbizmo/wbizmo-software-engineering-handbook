# Lesson 01 Cheatsheet: Introduction To CSS

CSS:
    Cascading Style Sheets.

Purpose:
    Style HTML.

Rule:

    selector {
      property: value;
    }

Example:

    p {
      color: red;
    }

Inline CSS:

    <p style="color: red;">Text</p>

Internal CSS:

    <style>
      p {
        color: red;
      }
    </style>

External CSS:

    <link rel="stylesheet" href="styles.css">

Best practice:
    Use external CSS for real projects.
