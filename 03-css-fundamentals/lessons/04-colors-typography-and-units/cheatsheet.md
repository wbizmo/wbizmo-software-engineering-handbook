# Lesson 04 Cheatsheet: Colors, Typography and Units

Named color:

    color: red;

Hex:

    color: #ff0000;

RGB:

    color: rgb(255, 0, 0);

RGBA:

    color: rgba(255, 0, 0, 0.5);

Font family:

    font-family: Arial, sans-serif;

Font size:

    font-size: 16px;

Font weight:

    font-weight: 700;

Line height:

    line-height: 1.6;

Units:

    px
    %
    rem
    em
    vw
    vh

Key takeaway:
    Typography and spacing decide how professional a page feels.
