# Lesson 05 Cheatsheet: Forms, Semantic HTML and Accessibility

Form:

    <form></form>

Label:

    <label for="email">Email</label>

Input:

    <input id="email" name="email" type="email">

Textarea:

    <textarea name="message"></textarea>

Select:

    <select name="country">
      <option value="ng">Nigeria</option>
    </select>

Button:

    <button type="submit">Submit</button>

Semantic Elements:

    header
    nav
    main
    section
    article
    aside
    footer

Accessibility Basics:

- Use labels.
- Use alt text.
- Use logical headings.
- Use meaningful link text.
- Use buttons for actions.
- Use links for navigation.

SEO Basics:

- Clear title.
- Logical headings.
- Semantic structure.
- Good alt text.
- Meaningful content.

Key takeaway:
    Good HTML is meaningful, accessible, and structured.
