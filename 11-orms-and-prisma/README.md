# ORMs & Prisma

## Overview

This module teaches Object Relational Mappers (ORMs) and Prisma.

Students will learn how modern backend applications communicate with relational databases without writing raw SQL for every operation.

Prisma is one of the most widely used ORMs in the TypeScript ecosystem and is heavily used in production Fastify, Express, and Node.js applications.

## Why This Module Matters

Most professional backend systems use an ORM.

Benefits include:

- Faster development
- Better maintainability
- Type safety
- Safer queries
- Easier migrations
- Better developer experience

Prisma is a core part of many modern backend stacks.

## Learning Outcomes

By the end of this module you will be able to:

- Explain what an ORM is
- Explain Prisma architecture
- Create Prisma schemas
- Create models
- Run migrations
- Perform CRUD operations
- Manage relationships
- Implement pagination
- Build production-ready database layers
