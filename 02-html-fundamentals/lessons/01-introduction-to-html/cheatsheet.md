# Lesson 01 Cheatsheet: Introduction To HTML

HTML:
    HyperText Markup Language.

Purpose:
    Structure webpage content.

CSS:
    Style and layout.

JavaScript:
    Behavior and interactivity.

Tag:
    HTML syntax.

Element:
    Opening tag + content + closing tag.

Attribute:
    Extra information on an element.

Paragraph:

    <p>Hello</p>

Heading:

    <h1>Main Heading</h1>

Link:

    <a href="https://example.com">Visit</a>

Image:

    <img src="image.jpg" alt="Description">

Key takeaway:
    HTML gives meaning and structure to webpage content.
