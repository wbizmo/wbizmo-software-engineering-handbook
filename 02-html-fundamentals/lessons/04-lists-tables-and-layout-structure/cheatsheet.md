# Lesson 04 Cheatsheet: Lists, Tables and Layout Structure

Unordered list:

    <ul>
      <li>Item</li>
    </ul>

Ordered list:

    <ol>
      <li>Step</li>
    </ol>

Table:

    <table>
      <tr>
        <th>Heading</th>
      </tr>
      <tr>
        <td>Data</td>
      </tr>
    </table>

table:
    Table.

tr:
    Table row.

th:
    Heading cell.

td:
    Data cell.

div:
    Generic block container.

span:
    Generic inline container.

Key takeaway:
    Use lists for items and tables for data.
