# Lesson 04 Answers: Lists, Tables and Layout Structure

1. ul creates an unordered list.
2. ol creates an ordered list.
3. li creates a list item.
4. table creates a table.
5. tr means table row.
6. th represents a table heading cell.
7. td represents a table data cell.
8. No. Tables should be used for tabular data, not page layout.
9. div is a generic block container.
10. span is a generic inline container.
