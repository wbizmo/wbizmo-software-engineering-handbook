# Lesson 03 Cheatsheet: Box Model, Spacing and Sizing

Box model:

    Content
    Padding
    Border
    Margin

Padding:
    Space inside.

Margin:
    Space outside.

Border:
    Surrounds element.

Global box sizing:

    * {
      box-sizing: border-box;
    }

Margin shorthand:

    margin: 20px;
    margin: 10px 20px;
    margin: 10px 20px 30px 40px;

Padding shorthand:

    padding: 20px;
    padding: 10px 20px;

Center block:

    width: 300px;
    margin: 0 auto;

Key takeaway:
    Spacing is one of the biggest parts of clean UI.
