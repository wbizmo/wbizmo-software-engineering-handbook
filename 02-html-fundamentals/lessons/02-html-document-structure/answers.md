# Lesson 02 Answers: HTML Document Structure

1. It tells the browser to use modern HTML standards mode.
2. The html element contains the entire page.
3. The lang attribute describes the page language.
4. The head contains metadata, title, CSS links, SEO tags, and document information.
5. The body contains visible page content.
6. The title controls the browser tab title and helps SEO.
7. charset tells the browser how to interpret characters.
8. The viewport tag helps pages display correctly on mobile devices.
9. Nesting means placing elements inside other elements.
10. Indentation makes HTML easier to read and maintain.

## Exercise Solution

    <!DOCTYPE html>
    <html lang="en">
      <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>My Practice Page</title>
      </head>
      <body>
        <h1>Welcome</h1>
        <p>This is my first paragraph.</p>
        <p>This is my second paragraph.</p>
      </body>
    </html>
