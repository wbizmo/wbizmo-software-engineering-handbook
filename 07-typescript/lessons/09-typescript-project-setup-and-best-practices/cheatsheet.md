# Lesson 09 Cheatsheet: TypeScript Project Setup and Best Practices

Install:

    npm install --save-dev typescript

Initialize:

    npx tsc --init

Compile:

    npx tsc

Watch:

    npx tsc --watch

Build script:

    "build": "tsc"

Important setting:

    "strict": true

Folders:

    src/
    dist/
    types/
    services/

Key takeaway:
    Use strict mode and organize your code properly.
