# HTML Fundamentals Module Cheatsheet

HTML:
    HyperText Markup Language.

Purpose:
    Structure webpage content.

Starter:

    <!DOCTYPE html>
    <html lang="en">
      <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
      </head>
      <body>
      </body>
    </html>

Text:

    <h1>Main Heading</h1>
    <p>Paragraph</p>
    <strong>Important</strong>
    <em>Emphasis</em>

Links:

    <a href="https://example.com">Visit</a>

Images:

    <img src="image.jpg" alt="Description">

Lists:

    <ul>
      <li>Item</li>
    </ul>

    <ol>
      <li>Step</li>
    </ol>

Tables:

    <table>
      <tr>
        <th>Heading</th>
      </tr>
      <tr>
        <td>Data</td>
      </tr>
    </table>

Forms:

    <form>
      <label for="email">Email</label>
      <input id="email" name="email" type="email">
      <button type="submit">Submit</button>
    </form>

Semantic Elements:

    header
    nav
    main
    section
    article
    aside
    footer

Accessibility:

- Use alt text.
- Use labels.
- Use logical headings.
- Use meaningful links.

Key takeaway:
    HTML should be meaningful before it is beautiful.
