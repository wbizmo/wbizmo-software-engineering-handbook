# HTML Fundamentals Fun Facts

## Fun Fact 1

HTML was created by Tim Berners-Lee, the inventor of the World Wide Web.

## Fun Fact 2

The first public website was published at CERN.

## Fun Fact 3

HTML is not a programming language. It is a markup language.

## Fun Fact 4

Browsers are very forgiving. They often try to fix broken HTML automatically.

## Fun Fact 5

Semantic HTML can improve accessibility and SEO without adding any JavaScript.

## Fun Fact 6

The alt attribute is not just for SEO. It is essential for users who rely on screen readers.
