# Lesson 01 Answers: Introduction To HTML

1. HTML means HyperText Markup Language.
2. HTML structures webpage content.
3. CSS styles and lays out webpages.
4. JavaScript adds behavior and interactivity.
5. A tag is HTML syntax such as <p> or </p>.
6. An element usually includes an opening tag, content, and a closing tag.
7. An attribute adds extra information to an element.
8. href tells a link where to go.
9. alt describes an image for accessibility and fallback text.
10. HTML is important because screen readers and assistive technologies use semantic structure to help users understand content.

## Exercise Solutions

Simple page:

    <!DOCTYPE html>
    <html>
      <head>
        <title>Practice Page</title>
      </head>
      <body>
        <h1>My First Page</h1>
        <p>I am learning HTML.</p>
        <p>This is another paragraph.</p>
        <a href="https://github.com">Visit GitHub</a>
        <img src="photo.jpg" alt="A sample photo">
      </body>
    </html>
