# Lesson 03 Cheatsheet: Text, Links and Images

Headings:

    <h1>Main</h1>
    <h2>Section</h2>
    <h3>Subsection</h3>

Paragraph:

    <p>Text</p>

Strong:

    <strong>Important</strong>

Emphasis:

    <em>Emphasized</em>

Link:

    <a href="https://example.com">Visit</a>

New tab link:

    <a href="https://example.com" target="_blank" rel="noopener noreferrer">Visit</a>

Image:

    <img src="photo.jpg" alt="Description">

Absolute URL:
    Full address.

Relative URL:
    Path inside current project.

Key takeaway:
    Use HTML elements for meaning, not just appearance.
