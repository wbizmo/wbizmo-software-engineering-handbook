# CSS Fundamentals Fun Facts

## Fun Fact 1

CSS was first proposed in 1994.

## Fun Fact 2

Before CSS became common, many websites used tables for layout.

## Fun Fact 3

The cascade is the reason CSS can feel powerful and confusing at the same time.

## Fun Fact 4

Flexbox was created to solve many layout problems that were painful before modern CSS.

## Fun Fact 5

CSS Grid allows complex two-dimensional layouts without relying on external frameworks.

## Fun Fact 6

A professional-looking website often depends more on spacing and typography than flashy effects.
