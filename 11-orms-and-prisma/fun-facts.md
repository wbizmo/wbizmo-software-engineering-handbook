# ORMs and Prisma Fun Facts

1. Prisma generates fully typed database clients.
2. Prisma is one of the fastest-growing TypeScript ORM ecosystems.
3. Many modern SaaS products use Prisma.
4. Prisma supports PostgreSQL, MySQL, SQLite and more.
5. Strong typing prevents many database bugs before runtime.
6. Prisma makes refactoring much safer than raw SQL-heavy codebases.
