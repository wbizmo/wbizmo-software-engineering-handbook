# Lesson 01 Cheatsheet: Introduction To TypeScript

TypeScript:
    JavaScript with static typing.

File extension:
    .ts

Compile:
    Convert TypeScript to JavaScript.

Compiler:
    tsc

Example:

    function greet(name: string) {
      return "Hello, " + name;
    }

Variable:

    const age: number = 25;

Type object:

    type User = {
      id: string;
      email: string;
    };

Key takeaway:
    TypeScript helps catch mistakes before code runs.
