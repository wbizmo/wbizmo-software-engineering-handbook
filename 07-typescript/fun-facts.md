# TypeScript Fun Facts

1. TypeScript was created by Microsoft.
2. TypeScript was first released in 2012.
3. TypeScript powers many large-scale applications.
4. React, Angular and many Node.js projects use TypeScript heavily.
5. TypeScript compiles to JavaScript.
6. VS Code is largely written in TypeScript.
7. Many modern backend frameworks recommend TypeScript.
8. TypeScript has become one of the most loved developer technologies.
