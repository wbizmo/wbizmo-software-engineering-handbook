# Lesson 06 Cheatsheet

CRUD:

    Create
    Read
    Update
    Delete

Routes:

    GET /books
    GET /books/:id
    POST /books
    PUT /books/:id
    DELETE /books/:id

Status:

    200 OK
    201 Created
    204 No Content
    400 Bad Request
    404 Not Found
    500 Server Error

REST rule:
    Use nouns, not verbs.
